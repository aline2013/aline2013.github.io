评论系统

  • 目前评论系统有多说，网易云跟帖，disqus等，但是多说貌似最多人用，但是今年6月多说网站关闭了，于是我选择了disqus。

配置Disqus过程

  1. 到Disqus官网下，先注册账号，登录后进入home,点击左上角下拉

  2. 进入页面后，跳转后拉到页面最下方，点GET STARTED, 再次跳转后选择 I want to install Disqus on my site

  3. 在新页面中填写你的website name、type和language,比如我的

  4. 创建成功后，在新页面点击Install Disqus，选择Platform时，直接选择拉到页面最下面，点击按

  5. 又是拉到底部，点击Configure，URL设置为你的Github page地址
    ，这里disqus网站的初步配置即完成
